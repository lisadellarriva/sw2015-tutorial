About the application

The application is about Sandwiches!

Since the ontology used is quite small and limited in terms of both sandwich types(classification) and instances, one might get the sense that the application lacks on pratical uses. However, it was of great learning value!

Application's features:
  • You can explore the ontology via SPARQL queries
  • You can enable/disable the reasoner as you wish for the searches
  • You can add new sandwich to the ontology (note: not running reasoner!)
  • You can see the existing sandwiches filtered by characteristics (note: always running reasoner!)
